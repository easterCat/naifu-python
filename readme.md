# 版本

使用flask做prompt工具网站的后台

- python 3.8.15

```commandline
python -m venv tag_venv 
source ./tag_venv/bin/activate
pip install -r requirements.txt
```

## 创建requirements.txt

- 使用freeze

```commandline
pip freeze > requirements.txt .
```

- 使用pipreqs

```commandline
pip install pipreqs
pipreqs ./ --force
```
